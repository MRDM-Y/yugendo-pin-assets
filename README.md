# yugendo-pin-assets

Image host for **YugenDoJapan** (Etsy Japanese digital-art shop) Pinterest pins.

## Sole purpose

This repository exists for **one purpose only**: hosting Pinterest creative
images so their `raw.githubusercontent.com` URLs can be referenced as the
`Media URL` column of Pinterest **bulk-upload CSVs**.

Pinterest fetches each image **once** at CSV-import time and re-hosts it on its
own CDN (`pinimg`); thereafter the GitHub URL is no longer served. Traffic to
this repo is therefore a one-time, few-MB-per-week trickle — it is **not** used
as an ongoing CDN.

It is **not** used for source code, documents, releases, or any other purpose.

## Contents

- `creatives/` — Pinterest 2:3 framed-mockup images (1000×1500), one file per
  pin. Files are framed-in-room marketing mockups only (never the clean
  full-resolution artwork). Generated by `tools/pinterest/creative.py` in the
  private `PoD` repo and pushed here by `tools/pinterest/drip.py export-csv`.
